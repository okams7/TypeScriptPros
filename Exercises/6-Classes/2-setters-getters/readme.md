# اكتب برنامجًا باستخدام TypeScript يقوم بما يلي:

أنشئ فئة (Class) باسم Book تحتوي على الخصائص التالية كمجالات خاصة (Private):

_title (اسم الكتاب) نوعها string.

_author (مؤلف الكتاب) نوعها string.

_publisher (الناشر) نوعها string.

_isbn (رقم الكتاب المميز) نوعها string.

قم بإضافة مُنشئ (Constructor) لاستقبال القيم الأولية لهذه الخصائص وتعيينها.

أضف دوال Getter لكل خاصية لإرجاع قيمها.

أضف دوال Setter لكل خاصية لتحديث القيم.

أضف خاصية Setter باسم fullDetails لتحديث جميع القيم دفعة واحدة عبر سلسلة نصية مفصولة بفواصل (على سبيل المثال: "Title,Author,Publisher,ISBN").

إذا لم يتم تمرير السلسلة بصيغة صحيحة (أي لم تحتوي على 4 أجزاء مفصولة بفواصل)، قم برمي خطأ مناسب.

قم بإنشاء كائن جديد من الفئة، و:

اطبع اسم المؤلف (author) باستخدام الـ Getter.

عدل اسم الناشر (publisher) باستخدام الـ Setter.

استخدم الخاصية fullDetails لتحديث كافة القيم.

اطبع تفاصيل الكتاب باستخدام الـ Getter لجميع الحقول.

المخرجات المتوقعة:

يتم طباعة اسم المؤلف الأصلي.

يتم تحديث اسم الناشر وطباعته.

يتم تحديث كافة تفاصيل الكتاب دفعة واحدة وطباعتها باستخدام الحقول.
